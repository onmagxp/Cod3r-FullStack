`Const` é usado no lugar de ´var´ ou ´let´ para dizer que aquela variável é constante, ou seja, é inalterável. É recomendável começar o código com `const` para identificar os nomes das variáveis caso ache que não vai precisar alterar ao decorrer do algoritmo. É melhor alterar para ´var´ ou ´let´ se necessário do que ao contrário, e também é recomendável ter mais `const`  do que `var` no código.
